# CSS3 Animation Gallery | CSS3 动画效果展示

一个基于 Vue 3 + Vite 构建的 CSS3 动画效果集合项目，致力于展示丰富多样的 CSS3 动画效果。

## ✨ 特性

- 🚀 基于 Vue 3 + Vite 构建，性能优异
- 💡 纯 CSS3 实现，无需额外动画库
- 📦 组件化设计，即插即用
- 🎨 代码简洁，易于理解和使用
- 🔄 持续更新，不断丰富动画效果库

## 🎯 动画效果列表

- 圣诞树动画
- 更多动画持续更新中...

## 🛠️ 技术栈

- Vue 3
- Vite
- CSS3
- JavaScript
